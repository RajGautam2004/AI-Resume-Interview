## 🆘 If Something Goes Wrong

```
PROBLEM                          SOLUTION
═══════════════════════════════════════════════════════════════

"npm not found"                  Install Node.js from nodejs.org

"Port 5000 in use"               Change PORT in .env file
                                Or kill process: lsof -ti:5000 | xargs kill

"Cannot find module"             npm cache clean --force
                                npm install (again)

"Webcam not working"             1. Allow browser permissions
                                2. Refresh page
                                3. Try different browser
                                4. Check webcam connected

"CORS error"                     1. Backend must be running
                                2. Check VITE_API_URL in .env
                                3. Restart both frontend & backend

"Face detection slow"            Check GPU available (F12 console)
                                Performance depends on hardware

"Tests failing"                  npm cache clean --force
                                delete node_modules
                                npm install (fresh)

"Can't connect to API"           1. npm run dev:backend (Terminal 1)
                                2. npm run dev:frontend (Terminal 2)
```
